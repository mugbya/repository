## 使用

### 安装依赖

    pip install -r requirements.txt

### 建立数据模型

    python3  manage.py syncdb

### 创建系统管理者（首次不用执行）

     python3 manage.py createsuperuser

### 运行服务器

    python3 manage.py runserver 0.0.0.0:8000
